## Lessons Learned

- Thorough graph QA takes longer than expected across environments.
- Cross-checking render consistency (local markdown, GitHub preview, VSCode preview) is critical for professional presentation.
- Final path references and image embedding require extra care to avoid last-minute issues.
- Clear build logs support transparency and protect authorship, especially in open science settings.
